# 🚀 销售 AI 闭环系统 —— v0.0.2 需求文档（补齐版）

## 一、文档目标
在`v0.0.1`和`v0.0.2`文档的自测中，发现需求缺失的地方进行补充实现

## 二、功能需求

1. 左侧弹框的LLM配置，可以读取env文件进行配置，用户可以修改后存入数据库，LLM配置的优先级：数据库 > env默认配置
2. 右侧弹框增加v0.0.2的RAG 知识库前端相关能力，需要补充上传界面
3. v0.0.1版本中的将 PRD 转为前端代码字符串有问题，应该是使用LLM拼接prompt根据需求生成一份HTML文件（可以在本地运行的文件，不需要依赖其它文件，如果一定要依赖，可以使用cdn链接获取），然后在前端页面嵌入iframe演示这个前端h5文件
4. v0.0.1版本中的合同生成应该根据目前的prd文件 + 进行RAG相关内容的检索，通过LLM生成一份合同文件，需要补充LLM相关prompt和改善相关逻辑
